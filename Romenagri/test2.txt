_darasa_tee - _tarameema  saanacha_ha _dasa_tawaeejza _deekhee_aee

(1907 _taa 23 maaracha, 1931_ha) baraszagzeera kee ja_dauja_ha_da _aajzaa_dee kaa _heerau. bhaga_ta sanagha saushalasata _aenakzalaaba kaa _hzaamee _thaa. _tzabakzaa_ta saee paaka baraabaree kee sa_tza_hza para kzaayama _hana_dausa_taanee mayaashara_ha chaa_ha_taa _thaa. _jzala_aee laayala paura kaee mau_jza_aee banaga_ha maee_m pee_daa _hauyaee. kaamaa gaataa ja_haajza waalaee _aejee_ta sanagha _aena kaee chachaa _thaee. jalayaanawaala_ha baagza kza_tala yaama _oura _aee_dama _ta_aee_ouna kee _ta_hzareeka kaee khzaunee_m waakzayaa_ta saee _aeszzara kzabaula kayaa. 1921 maee_m _aesakaula chhau_rda _dee _oura neeshanala kaalaja maee_m _ta_aeeleema sharau_aee kee. 1927 maee_m laa_haura maee_m _dasa_hara_ha bama keesa kaee salasalaee maee_m garaphza_taara _hauyaee _oura shaa_hee kzala_aee_ha laa_haura maee_m rakhaeegayaee. _jzamaana_ta para ra_haayee kaee ba_aee_da naujawaana bhaara_ta sabhaa banaayee _oura phara _aenakzalaaba pasana_doa_m maee_m shaamala _hau gayaee. _da_halee maee_m yeena _aesa wakza_ta, jaba marakajzee _aesamabalee kaa _aejalaasa _hau ra_haa _thaa _ae_nzoa_m naee _oura baee kaee _da_ta naee _aesamabalee _haala maee_m _dhamaakaa pee_daa karanaee waalaa bama pheenakaa. _daunoa_m garaphza_taara karalee_aee gayaee. _aee_daala_ta naee _aeemara kzee_da kee sajzaa _dee.

1928 maee_m saayamana kameeshana kee _aama_da para laa_haura reelawaee _aesateeshana para jzabara_dasa_ta _ae_hza_tajaajee ma_jzzaa_hara_ha _hawaa. pauleesa naee laathee chaaraja kayaa jasa maee_m laala_ha laajapa_ta raayaee jzakhzamee _hau gayaee. _aesa wakza_ta laa_haura kaee seenayara saparateenadanata pauleesa masatara sakaata _thaee. _aenakzalaaba pasana_doa_m naee _aena koa _halaaka karanaee kaa manaszauba_ha banayaa. leekana _aeeka _dana pachhalaee pa_hara jaba masatara saanadarasa _aesasatanata saparateenadanata pauleesa laa_haura _aepanaee _daphza_tara saee mautara saayeekala para _daphza_tara saee nakalaee _tau shawya raama raaja garau _oura bhaga_ta sanagha wagzeera_ha naee _aena koa gaulee maarakara _halaaka kara _dayaa. _hzawaala_daara jeena naee sanagha kaa _tayaakzaba kayaa. _aena_hoa_m naee _aesa koa bhee gaulee maara _dee _oura dee _ai wee kaalaja _haasatala maee_m kapa_rdaee ba_dala kara mana_tashara _hau gayaee. _aakhzara khzaana ba_haa_dara sheekhza _aeeba_da _aela_aeejzeejza naee kashameera baladanaga laa_haura saee _aeeka raa_ta _tamaama _aenakzalaaba pasana_doa_m koa garaphza_taara kara layaa. laa_haura kaee seenatarala jeela kaee kamara- _aee_daala_ta maee_m _aena para makza_dama_ha chalayaa gayaa. bhaga_ta sanagha _oura _da_ta _aesa saee kzabala _aesamabalee bama keesa maee_m sajzaa paa chakaee _thaee. makza_dama_ha _teena saala _taka chala_taa ra_haa. _hzakauma_ta kee _tzaraphza saee khzaana szaa_hzaba kzalana_dara _aeelee khzaana _oura malajzamaana kee _tzaraphza saee laala_ha _aemara_da _aesa seenayara wakeela _thaee. bhaga_ta sanagha _oura sakha _dawya koa sajzaayaee mau_ta kaa _hzakama _dayaa gayaa _oura 23 maaracha, 1931 koa _aena koa phaanasee _daee _dee gayee. phzeeraujza paura kaee kzareeba _darayaayaee sa_talaja kaee kanaaraee, _aena kee laashoa_m koa jalaa _dayaa gayaa. ba_aee_da maee_m ya_haa_m _aena kee yaa_dagaara kzaayama kee gayee.
